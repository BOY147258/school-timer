# ⚡ 极速计时 - 校园体育计时器

专为中小学体育课、校运会设计的极简计时器。

## 🌐 在线访问

**👉 https://boy147258.github.io/school-timer/**

## ✨ 功能特点

- 🎯 **极简界面** - 深色主题，一目了然
- 🔫 **专业发令音效** - 模拟真实赛场（各就位→预备→枪声）
- 📏 **多种距离** - 支持60/100/200/400/800/1000米
- 🏃 **多道次设置** - 1-8道可选
- 🎤 **多种发令方式** - 手动/声音识别/倒计时
- 📱 **PWA支持** - 可安装到手机桌面，离线使用
- 📊 **成绩导出** - CSV格式，方便记录分析

## 🚀 快速开始

### 发令端使用
1. 打开页面，选择「发令端」
2. 设置比赛距离和道次
3. 选择发令方式（推荐「手动」）
4. 点击「开始计时」
5. 按屏幕提示操作（各就位→预备→点击结束）

### 终点端使用（可选）
1. 打开页面，选择「终点端」
2. 输入发令端显示的房间码，连接设备
3. 摄像头对准终点线
4. 发令后终点端自动识别冲线

## 📋 使用场景

- 🏫 中小学体育课计时训练
- 🎪 校运会田径比赛
- 🏋️ 课后训练成绩记录

## 🛠 技术栈

- HTML5 + CSS3 + JavaScript
- Web Audio API（发令音效）
- MediaDevices API（终点识别）
- WebRTC（设备连接）
- PWA（离线支持）
